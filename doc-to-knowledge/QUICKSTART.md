# 下载后快速上手指南

> **面向人群**：不会写代码也能用！只需要改几个配置文件就行。
>
> **这个工具能做什么**：把你的文档（飞书文档 / Word / Markdown 笔记）自动拆解成一条一条的"知识点"，存到飞书多维表格里，方便你后续做内容运营。

---

## 先搞懂：这个工具是给谁用的？

这个工具**出厂默认**是面向这类场景的：

| 项目 | 默认设定 |
|------|---------|
| **目标受众** | 月预算 5千~5万 的中小商家、个体创业者、小电商卖家 |
| **内容来源** | 营销类视频课程的转录稿 |
| **提取视角** | 站在"小白商家"角度，把专业术语翻译成大白话 |
| **痛点标签** | 预算有限、缺乏经验、人手不足、获客困难、转化率低、不懂数据、素材匮乏、复购难 |
| **AI 模型** | GPT-4o（也可以换成其他模型） |

**如果你的受众、行业、场景不同，完全可以改**（后面会教你怎么改）。

## 第 1 步：填两个配置文件（必做，5 分钟）

下载后你会看到两个模板文件，需要复制并填入你自己的信息。

### 1.1 创建 `.env` 文件 — 填密钥

把 `.env.example` 复制一份，改名为 `.env`，然后打开它，填 3 个东西：

```
FEISHU_APP_ID=cli_你的飞书应用ID
FEISHU_APP_SECRET=你的飞书应用密钥
OPENAI_API_KEY=sk-你的AI模型密钥
```

| 要填什么 | 去哪里拿 |
|---------|---------|
| `FEISHU_APP_ID` | 登录 [飞书开放平台](https://open.feishu.cn/app) → 创建自建应用 → 凭证页面 |
| `FEISHU_APP_SECRET` | 同上，和 App ID 在同一个页面 |
| `OPENAI_API_KEY` | 如果用 OpenAI：去 [platform.openai.com](https://platform.openai.com/api-keys) 创建；如果用其他 AI，填对应的 key |

> 详细的飞书应用创建教程见 `references/SETUP_GUIDE.md`

### 1.2 创建 `config.json` 文件 — 填表格地址

把 `config.example.json` 复制一份，改名为 `config.json`，然后打开它，填 2 个东西：

```json
{
  "feishu": {
    "app_token": "你的多维表格Token",
    "table_id": "你的数据表ID"
  }
}
```

**怎么找这两个值？** 打开你的飞书多维表格，看浏览器地址栏：

```
https://你的域名.feishu.cn/base/BascnXXXXXX?table=tblYYYYYY
                                   ↑ 这是 app_token    ↑ 这是 table_id
```

## 第 2 步：在飞书里建好表格（必做，10 分钟）

在飞书中新建一个多维表格，添加以下 13 列：

| 列名 | 类型 | 说明 |
|------|------|------|
| 知识点ID | 文本 | 自动生成，不用管 |
| 来源文档 | 文本 | 自动填入 |
| 来源章节 | 文本 | 自动填入 |
| 知识点标题 | 文本 | AI 自动生成的 15 字标题 |
| 知识点类型 | 单选 | 选项填：`概念解释` `操作方法` `避坑提醒` `数据洞察` `行业趋势` `工具技巧` |
| 核心内容 | 文本 | AI 生成的 200~500 字描述 |
| 关键要点 | 文本 | 要点列表 |
| 适用场景 | 文本 | 适合什么情况用 |
| 痛点标签 | 多选 | 选项填：`预算有限` `缺乏经验` `人手不足` `获客困难` `转化率低` `不懂数据` `素材匮乏` `复购难` |
| 难度等级 | 单选 | 选项填：`入门` `进阶` `高级` |
| 原文摘录 | 文本 | 原始文段摘录 |
| 处理状态 | 单选 | 选项填：`待审核` `已通过` `已弃用` |
| 社交文案状态 | 单选 | 选项填：`未生成` `已生成` `已发布` |

**别忘了！** 建完表后，要把你的飞书自建应用加为表格的「可编辑」协作者，否则工具写不进去。

---

## 第 3 步：跑起来（必做，2 分钟）

在终端（命令行）里执行：

```bash
# 安装依赖（只需要第一次）
pip3 install -r requirements.txt

# 先用预览模式试一下（不会真的写入飞书，放心）
python3 scripts/pipeline.py --doc-token 你的文档Token --dry-run

# 没问题的话，正式跑
python3 scripts/pipeline.py --doc-token 你的文档Token
```

> **文档 Token 怎么找？** 打开飞书文档，看地址栏：
> `https://你的域名.feishu.cn/docx/XxXxXxXx` ← 最后这段就是 Token

## 如果你的受众/行业/场景不同，改这里（可选）

这是最关键的自定义部分——你不需要会写代码，只需要会改文字。

### 改目标受众和提取视角

打开 `scripts/knowledge_extractor.py`，找到大约第 80 行的这段文字：

```
你是一位数字营销领域的资深内容策略师，专门服务中小客商家 (SMB)。
```

以及下面的「目标受众画像」部分。**直接改成你的受众就行**，比如：

| 你的场景 | 改成什么 |
|---------|---------|
| 面向程序员的技术教程 | "你是一位技术内容专家，专门为初级开发者整理学习笔记" |
| 面向宝妈的育儿内容 | "你是一位母婴领域的内容策略师，专门服务新手妈妈" |
| 面向餐饮老板的运营课 | "你是一位餐饮行业的运营顾问，专门帮小餐饮店提升营业额" |
| 面向医生的学术文献 | "你是一位医学内容编辑，专门为临床医生整理文献要点" |

**核心原则**：告诉 AI "你是谁"、"你服务谁"、"对方关心什么"，AI 就会按你的视角来提取和改写。

### 改痛点标签

同一个文件里，找到大约第 29 行的 `VALID_PAIN_TAGS`：

```python
VALID_PAIN_TAGS = [
    "预算有限", "缺乏经验", "人手不足", "获客困难",
    "转化率低", "不懂数据", "素材匮乏", "复购难",
]
```

直接换成你的行业痛点就行。记得飞书多维表格的「痛点标签」多选项也要同步改。

### 改 AI 模型

打开 `config.json`，找到 `ai` 部分：

```json
"ai": {
    "model": "gpt-4o"
}
```

可以换成：`gpt-4o-mini`（更便宜）、`gemini-2.5-flash`（免费额度高）等。

如果你在国内访问不了 OpenAI，可以在 `.env` 中加一行代理地址：

```
OPENAI_BASE_URL=https://你的代理地址/v1
```

## 常见问题

**Q：我不懂代码，能用吗？**
A：能！你只需要改配置文件里的文字，不需要写任何代码。上面标注了具体改哪里、改成什么。

**Q：提示"获取 tenant_access_token 失败"**
A：飞书的 App ID 或 App Secret 填错了，回去检查一下。

**Q：提示"权限不足"**
A：飞书文档/多维表格没有授权给你的应用。打开文档 → 右上角「…」→ 添加文档应用。

**Q：AI 什么都没提取出来**
A：检查 AI 的 API Key 是否有效、是否有余额。可以试试 `--model gpt-4o-mini` 用便宜模型先测试。

**Q：我想处理本地的 Word 文件**
A：用 `--local-docx` 参数：`python3 scripts/pipeline.py --local-docx 你的文件.docx --dry-run`

**Q：我想处理 Markdown 笔记**
A：用 `--local-md` 参数：`python3 scripts/pipeline.py --local-md 你的笔记目录/ --dry-run`
